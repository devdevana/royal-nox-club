# Royal Nox Club 🌿
Uma page moderna de um clube esportivo premium, com foco em bem-estar, atividades e experiência visual sofisticada.

## ✨ Sobre o projeto

O **Royal Nox Club** é um site desenvolvido com React que apresenta um clube completo com diversas atividades físicas e serviços voltados para qualidade de vida.

O projeto foi pensado com foco em:

* design elegante
* boa experiência do usuário
* organização de layout

## 🚀 Tecnologias utilizadas

* React
* Vite
* Tailwind CSS


## 🎨 Funcionalidades

* Seção "Sobre nós"
* Cards de atividades com efeito hover
* Footer com formulário de contato
* Navegação com scroll suave

# 📁 Estrutura do projeto

```bash
src/
  components/
    Header.jsx
    Hero.jsx
    About.jsx
    Servicos.jsx
    Footer.jsx
  App.jsx
```
